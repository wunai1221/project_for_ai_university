# Frontend 前端模組（A 同學負責）

## 負責範圍
本資料夾為網站的前端介面，負責使用者與機器人互動的畫面呈現。

## 主要功能（預計）
- 使用者輸入問題的對話介面
- 顯示機器人回答的訊息區塊
- 網站整體版面與風格設計

## 與其他模組的串接
- 透過 API 呼叫 Backend，將使用者問題送出並接收回答
- API 格式規範請參考 `/shared/api_spec.md`
- **不需要直接接觸 RAG 或向量資料庫**，所有資料查詢由 Backend 處理
